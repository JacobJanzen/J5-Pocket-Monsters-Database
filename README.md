## How to run
### Linux and macOS
```
.../3380_Project $ ./run-unix.sh
```
Press CTRL+C to close the the front-end and then CTRL+C again to close the back-end.
### Windows
```
...\3380_Project>run-windows.bat
```
Press CTRL+C in each cmd window to close the server.
